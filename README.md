## 🏃 Getting Started

Make sure you have **Node.js** installed. You can use [nvm](https://github.com/nvm-sh/nvm#installing-and-updating) to manage Node versions.

```sh
# 1. Clone the repository
git clone <YOUR_GIT_URL>

# 2. Navigate into the project directory
cd <YOUR_PROJECT_NAME>

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
```

The app will be live at `http://localhost:5173` with hot-module reloading.
